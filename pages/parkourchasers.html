Parkour Chasers

ABOUT_ME
Code is under Digital Bros Game Academy ownership

What is it?

Midcore 1v1 mobile game mixing the new emerging sport "Chase Tag" with a cartoon atmosphere. Players will earn points by tagging or running away from their opponent respectively until time runs out usings skills, avoiding obstacles and using the environment to their advantage

When was it made?

Mid 2023 as my third DBGA project. Developed in 2 month.

Who made it?

A team of 14 people:
  • 4 Game Designers
  • 4 Game Programmers (including me as Lead)
  • 2 Concept Artists
  • 4 3D Artists

What did I learn/practice?

  • Improvements in communication and managing skills as a Lead programmer in a medium team
  • Improvements in understanding the differences between programming for mobile
  • Tried to create a custom Events System with the aim of having a completely decoupled code
  • Usage of Unity Assemblies in order to have separate modules (again for the decoupled code aim)
  • Usage of FMOD for the Music/SFX
  • Creation of custom on screen input controller, management of multiple input controllers
  • Cinemachine

Escape Room

ABOUT_ME
Code is under Digital Bros Game Academy ownership

What is it?

2 levels puzzle game in which players have to find clues and solve puzzles in order to escape from the room

When was it made?

Mid 2023 as my fourth DBGA project. Developed in 24/30 hours as a mini internal game jam (spread over several days).

Who made it?

A team of 7 people:
  • 3 Game Designers
  • 2 Game Programmers (including me as Lead)
  • 2 3D Artists

What did I learn/practice?

  • Clean code architecture organized in modules
  • Improved usage of FMOD and creation of a custom system for triggering sounds
  • Cinemachine
